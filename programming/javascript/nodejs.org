#+TITLE: Node.js

* =npm=
** =init=

初始化 =package.json= 。

#+begin_src sh
npm init
#+end_src

** =install=

下载 =package.json= 中声明的依赖模块到 =node_modules= 。生成 =package-lock.json= ，用于锁定依赖的确切版本。

#+begin_src sh
npm install <options> <package>
  -g          # Install globally
  --save-dev
  --save-prod
#+end_src

** =run=

#+begin_src sh
npm run <command>
    start # 运行开发版本
    test  # 运行测试
        -- --coverage # 直接返回结果，不等待输入
    build # 编译生产版本
#+end_src
