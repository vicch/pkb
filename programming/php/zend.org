** Model

- Zend_Db 提供了面向对象风格的、数据库独立的读取数据方法；

** View

- 与 Smarty 等模板引擎不同，Zend 的模板是由 PHP 编写的；
- Zend_View 提供了 helper 系统，用于创建可复用的显示功能代码；
