* 辨析

| argument      | 实参（方法调用）                                     |
| parameter     | 形参（方法定义）                                     |
|---------------+------------------------------------------------------|
| attribute     | 特性（原本就具有的，如目录的特性）                   |
| property      | 属性（主动赋予的，与对象有关）                       |
|---------------+------------------------------------------------------|
| command       |                                                      |
| directive     |                                                      |
|---------------+------------------------------------------------------|
| call          | 调用（有意）                                         |
| invoke        | 激活？（可能非有意）                                 |
|---------------+------------------------------------------------------|
| function      | 函数（和对象无关）                                   |
| method        | 方法（和对象相关）                                   |
|---------------+------------------------------------------------------|
| overload      | 重载（同一个类中，方法名相同，参数不同）             |
| override      | 覆盖（继承关系中，函数签名相同，实现不同）           |
| overwrite     | 重写（基类函数被隐藏？）                             |
|---------------+------------------------------------------------------|
| parse         | 解析                                                 |
| resolve       | 解析                                                 |
|---------------+------------------------------------------------------|
| introspection | examine types and objects at runtime                 |
| reflection    | examine and manipulate (props, functions) at runtime |
|---------------+------------------------------------------------------|
| demo          |                                                      |
| dummy         |                                                      |
| fake          |                                                      |
| mock          |                                                      |
| pseudo        |                                                      |
|---------------+------------------------------------------------------|
| agnostic      |                                                      |
| independent   |                                                      |
| neutral       |                                                      |
