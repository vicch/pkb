* Pandas
** =pandas.DataFrame=

二维表格数据结构。

*** =pandas.DataFrame()=

#+begin_src python
pandas.DataFrame(
[   data
,   index
,   columns
,   dtype
,   copy
])
#+end_src

#+begin_src python
pandas.DataFrame([[1,2,3],[10,20,30]])
#    0  1  2
# 0  1  2  3
# 1 10 20 30

pandas.DataFrame([[1,2,3],[10,20,30]],index=["1st","2nd"])
#      0  1  2
# 1st  1  2  3
# 2nd 10 20 30

pandas.DataFrame([[1,2,3],[10,20,30]],columns=["a","b","c"])
#    a  b  c
# 0  1  2  3
# 1 10 20 30

pandas.DataFrame([{"a":1,"b":2},{"a":3,"c":4}])
#   a   b   c
# 0 1   2 NaN
# 1 3 NaN   4
#+end_src
