* Tasks
** Create Minimal Java Project

IntelliJ > Create new project:

- Type: Java
- Project SDK: Java 1.8
- Additional libraries and frameworks: (Empty)
- Create project from template: Command line app
- Base package: =com.<custom>=

** Add Library

Project structure > Libraries > New project library (the =+= button) > From Maven:

- Keyword: =org.<library>=
